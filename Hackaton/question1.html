Théorie: Premiers pas... Commençons avec la base de la création de site web : Le HTML est un langage de balisage destiné à représenter la structure d’une page: Titres, paragraphes, images, et autres sections. Le CSS vient se rajouter à la structure HTML et permet d'en modifier le visuel: taille, couleur, police et une multitude d'autres effets. Le PHP, quant à lui, est plutôt un langage de programmation dit « côté serveur », il n'est donc pas utilisé pour le design.

Avec quel langage peut-on modifier le design (couleurs, couleur de fond, …) d’un site web ?
