Lien HTML et CSS

Pour lier son code HTML avec son code CSS, il suffit de mettre un lien dans sa page HTML qui pointe vers la page de style. Pour savoir quel code de style s'applique à quel élément, nous utilisons des sélecteurs. Les sélecteurs #id et .class nous permettent de cibler un élément en particulier. Nous devons écrire nos attributs id et class à l’intérieur de la balise ouvrante d’un élément HTML et leur attribuer un nom qui sera repris dans la feuille de style CSS.

Sur base du code html suivant, tape le symbole adéquat dans le code css.

ID = # Class = .

Html :
<p id=«paragraphe » > Je suis un paragraphe </p>
Css :
paragraphe {Color : red ;}
