Théorie: Les structures conditionnelles
La structure conditionnelle est un concept essentiel en programmation. Grâce à lui, nous allons pouvoir exécuter l'une ou l'autre instruction selon le cas qui se présente à nous. Les conditions simples s'écrivent avec les mots anglais « if » et « else », qui signifient respectivement « si » et « sinon ». En français, cela donne : Si « telle condition » est vraie, alors on exécute « telle instruction » . Sinon, on exécute « telle autre instruction »

Si monAge vaut 18 ans, quel est le résultat du bout de code ci-dessous :
if (monAge >= 18){
print(« Bienvenue en boîte, tu es majeur.e ! ») ; }
Else {
Print(« Tu es mineur.e, brosse-toi les dents et au lit ! ») ; }
